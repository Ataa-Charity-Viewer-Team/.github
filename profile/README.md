### 🌟 Our Mission
An open-source development collective dedicated to engineering transparent, scalable, and AI-audited digital ecosystems for charity campaigns, donor relationships, and mobile donation tracking.

---

### 📂 Core Repositories & Systems

📱 **Ataa Care Flutter**: Cross-platform mobile client for donors.
🌐 **Ataa Charity Platform**: React 18 & TypeScript frontend client dashboard.
🎓 **Ataa Platform API**: Node.js & Express server featuring Llama/Gemma AI auditing.
